# Go-dashboard

A dashboard application built in Go.

## Features

Link saver (bookmark)
Kanban board
Twitter drafts

## 🚀 Showcase

[Link to showcase]()

## 📦 Installation

To install the necessary dependencies, run the following command:

```bash
go mod tidy
```

## ▶️ Running the application

To run the application, use the following command:

```bash
go run main.go
```
